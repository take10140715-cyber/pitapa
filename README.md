PiTaPa利用チェックツール v55
PiTaPa倶楽部からダウンロードした利用明細CSVを読み込み、**「定期区間との重複」や「休日利用」**を自動判定するWebアプリケーションです。 鉄道子会社のコンサルタント業務等における、通勤定期代の二重払いチェックや不適切な利用の監査を効率化します。

🚀 主な機能
自動重複判定: 登録された定期区間と、実際の利用経路を比較し、重複があれば自動アラート。

高度な経路探索: 事業者またぎ（例：大阪メトロ～阪急）や、相互直通運転を考慮した経路探索ロジックを搭載。

休日・深夜判定: 土日祝日および年末年始の利用を自動判別し、「休日」バッジを表示。

ゆらぎ吸収: CSV特有の略称（「大地下鉄」「神鉄」など）や、駅名の表記揺れをマスタデータで自動変換。

多機能リスト管理:

「未提出者」「社員未登録者」のリストアップ。

「確認OK」のトグル管理（ダブルクリック操作）。

外部連携: 重複区間のリンクをクリックするだけで「Yahoo!路線情報」の検索結果を別タブで表示。

出力機能: 印刷用A4縦・高密度レイアウト対応。解析後の全データCSVエクスポート機能。

設定保存: 登録した社員データや独自設定をCSVとして保存・再読込可能。

🛠 使い方
1. CSVの取込
[CSV取込] タブから、PiTaPa倶楽部の「ご利用代金明細（CSV）」を選択、またはフォルダごとドラッグ＆ドロップしてください。複数人・複数月分の一括読み込みが可能です。

2. 定期・設定の登録
[定期・設定] タブで、対象となる社員名と定期区間を登録します。

重要: 社員名はCSV内の「○○　○○　様ご利用分」と全角スペースを含めて一致させてください。

発駅と着駅で異なる事業者を設定できます。

3. 結果の確認
[結果確認] タブで集計結果を確認します。

要確認: 定期重複や休日利用があるレコードが強調表示されます。

ダブルクリック: 判定が問題ない場合、名前をダブルクリックすることで「確認OK」リストに移動できます。

4. 印刷・出力
解析結果を保存したい場合は、[結果CSV出力] を行うか、ブラウザの印刷機能を使用してPDFまたは紙で保存してください。

📊 技術仕様
Framework: React 18 (Standalone Babel)

Styling: Tailwind CSS

Algorithm: ダイクストラ法をベースとした簡易経路探索（駅間グラフ構造）

Date Format: ゼロ埋めの有無や区切り文字（/, -）を問わない正規化集計

Print: CSS Media Queriesを用いたA4縦専用設計
